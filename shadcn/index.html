tvPick Utility

Utility for picking and processing Tailwind Variants props

Open in

Code Example

import { tvPick } from "lib/tv-pick"
import { tv } from "tailwind-variants"

const buttonVariants = tv({
  base: "px-4 py-2 rounded",
  variants: {
    variant: {
      primary: "bg-blue-500",
      secondary: "bg-gray-500"
    }
  }
})

// Extract variant props and apply classes
const props = tvPick(buttonVariants, {
  variant: "primary",
  onClick: handleClick
})

Installation

1. Add this registry to your components.json 

// components.json
{
  "registries": {
    "@gkurt": "https://gkurt.com/shadcn/{name}.json"
  }
}
or 
bunx shadcn registry add @gkurt=https://gkurt.com/shadcn/{name}.json

2. Install components using the namespace 

npx shadcn@latest add @gkurt/poly
npx shadcn@latest add @gkurt/poly-astro
npx shadcn@latest add @gkurt/tv-pick

Or install directly via URL 

npx shadcn@latest add https://gkurt.com/shadcn/r/poly.json
