KurtGokhan Shadcn Registry

KurtGokhan's custom shadcn registry for various components and utilities.

Installation

1. Add this registry to your components.json 

// components.json
{
  "registries": {
    "@gkurt": "https://gkurt.com/shadcn/r/{name}.json"
  }
}

Or add it via the CLI

bunx shadcn registry add @gkurt=https://gkurt.com/shadcn/r/{name}.json

2. Install components using the namespace

npx shadcn@latest add @gkurt/poly
